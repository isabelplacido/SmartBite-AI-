Key Takeaways

  • With a small dataset, AlexNet can still learn useful patterns, but performance is limited.
  • Some classes are visually similar, which leads to confusion (e.g., similar packaging or colors).
  • More diverse, higher-quality data would likely improve generalization.
  • This project shows the potential of using image-based models as a layer of “food awareness” but it is not ready for real-world medical or nutrition decisions.
